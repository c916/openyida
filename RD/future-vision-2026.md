## 功能需求

### 页面目标
"未来视野2026"线上发布会高转化率报名页，通过紧迫感、社交证明、低摩擦表单和价值前置策略最大化报名转化率。

### 页面结构（8个区块）
1. **固定顶部栏** - 动态渐变背景，Logo + 实时倒计时 + CTA"立即锁定席位"
2. **Hero区** - 科技粒子动画背景，渐变主标题 + 双CTA + 实时报名动态轮播
3. **价值主张** - 3列玻璃态卡片（白皮书首发、顶级嘉宾、创新者社群）+ 数据背书
4. **议程时间轴** - 6个时间节点，垂直发光线条，悬停展开详情
5. **嘉宾阵容** - 5位嘉宾卡片（含1位神秘嘉宾剪影）
6. **社交证明** - Logo墙（灰度→彩色）+ 评价轮播 + 实时报名计数
7. **报名表单** - 4字段表单 + 席位进度条 + 提交动画
8. **成功状态** - Confetti动画 + 确认信息 + 日历下载 + 社交分享

### 交互逻辑
- 倒计时实时更新（天/时/分/秒翻转动画）
- 报名动态10秒自动轮换
- 评价轮播5秒自动切换
- 表单邮箱正则验证
- 提交调用宜搭 saveFormData API 存储数据
- 提交成功后显示 Confetti 庆祝动画
- 滚动触发 fade-up 动效
- Logo墙灰度悬停变彩色
